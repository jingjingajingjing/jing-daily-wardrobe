# 静静的智能衣橱

根据天气智能推荐穿搭，支持切换备选方案，不满意时还能自由拖拽搭配。页面美观即可，无特殊要求。

## 在线预览

[https://jing-daily-wardrobe.netlify.app/](https://jing-daily-wardrobe.netlify.app/)

## 功能需求

### 当前阶段
- **自动获取当天天气**：对接和风天气 API，获取实时天气
- **智能穿搭推荐**：根据天气推送适合今日的穿搭方案
- **切换选项**：支持在多个推荐方案之间切换浏览
- **自由搭配**：若都不满意，可自己拖拽组合搭配衣服

### 后续规划
- **定位**：支持地理位置自动获取
- **个人信息**：支持输入姓名
- **账号系统**：支持手机号注册/登录，不同账号存储各自数据（天气、位置、衣柜、穿搭偏好）

## 技术栈

- React 19 + TypeScript
- Vite 7
- Axios
- 和风天气 API

## 本地开发

```bash
npm install
npm run dev
```
