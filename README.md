# 静静的衣橱

根据天气智能推荐穿搭，支持切换备选方案，不满意时还能自由拖拽搭配。

## 功能需求

- **自动获取当天天气**：自动获取当日天气信息
- **智能穿搭推荐**：根据天气推送适合今日的穿搭方案
- **切换选项**：支持在多个推荐方案之间切换浏览
- **自由搭配**：若都不满意，可自己拖拽组合搭配衣服
- **页面**：美观即可，无特殊要求

## 技术栈

- HTML / CSS / JavaScript（当前）
- 天气 API（待接入）

## 本地开发

```bash
npm install
npm run dev
```

浏览器访问 http://localhost:3000
